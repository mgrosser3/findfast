# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-05

### Added

- Files can now be searched using glob patterns.
- Added a `--help` option to display usage information.

### Changed

- Improved command output using ANSI colors for better readability.

## [0.0.1] - 2025-11-17

### Added

- Introduced the command `ff PATTERN PATH`.
- Files can be searched using a regex pattern.
- The specified path is traversed recursively.
- Results display the files and the matching lines found.
